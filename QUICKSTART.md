# 快速开始指南

5 分钟上手多用户文件管理系统！

---

## 🚀 5 分钟快速启动

### 第 1 步：启动服务（1 分钟）

**Windows:**
```powershell
cd file_manager
start.bat
```

**Linux/Mac:**
```bash
cd file_manager
chmod +x start.sh
./start.sh
```

看到以下提示表示启动成功：
```
========================================
  多用户文件管理系统
========================================
访问地址：http://localhost:5000
默认管理员：admin / Admin@123
```

### 第 2 步：登录系统（1 分钟）

1. 浏览器打开：http://localhost:5000
2. 输入默认管理员账户：
   - 用户名：`admin`
   - 密码：`Admin@123`
3. 点击"登录"

### 第 3 步：修改密码（1 分钟）

⚠️ **首次登录必须修改密码！**

1. 点击右上角用户名 → "个人设置"
2. 输入当前密码和新密码
3. 点击"修改密码"

### 第 4 步：创建用户（1 分钟）

1. 点击顶部菜单 "用户管理"
2. 点击 "新建用户"
3. 填写信息：
   - 用户名：`zhangsan`
   - 密码：`User@123`
   - 确认密码：`User@123`
4. 点击 "创建"

### 第 5 步：分配权限（1 分钟）

1. 点击顶部菜单 "权限管理"
2. 选择刚创建的用户 `zhangsan`
3. 点击 "添加权限"
4. 填写：
   - 目录路径：`D:\Documents`（或 `/home/user/docs`）
   - 目录名称：`我的文档`
   - 权限级别：`读写`
5. 点击 "保存"

## 📁 开始使用

### 普通用户操作

1. **登录系统**
   - 使用分配的用户名和密码登录

2. **浏览文件**
   - 点击 "文件管理"
   - 点击有权限的目录

3. **上传文件**
   - 点击 "上传文件" 按钮
   - 选择文件（最大 1GB）
   - 等待上传完成

4. **下载文件**
   - 点击文件右侧的下载图标 ⬇️

5. **删除文件**
   - 点击文件右侧的删除图标 🗑️
   - 确认删除

6. **创建目录**
   - 点击 "新建文件夹"
   - 输入目录名称
   - 确认创建

7. **重命名**
   - 点击文件/目录右侧的重命名图标 ✏️
   - 输入新名称
   - 确认

### 管理员操作

#### 用户管理

- **创建用户**: 用户管理 → 新建用户
- **禁用用户**: 用户列表 → 点击禁用按钮
- **删除用户**: 用户列表 → 点击删除按钮
- **重置密码**: 用户列表 → 点击重置密码

#### 权限管理

- **授予权限**: 权限管理 → 选择用户 → 添加权限
- **撤销权限**: 权限管理 → 选择用户 → 点击撤销按钮
- **查看权限**: 权限管理 → 选择用户 → 查看详情

#### 审计日志

- **查看日志**: 审计日志 → 查看所有操作记录
- **筛选日志**: 使用搜索框按用户/操作类型筛选
- **导出日志**: 点击 "导出 CSV"（计划功能）

## 🎯 常用场景

### 场景 1：团队文件共享

1. 为每个团队成员创建账户
2. 创建共享目录：`D:\TeamShare`
3. 为所有成员授予该目录的读写权限
4. 团队成员上传/下载文件

### 场景 2：部门文档管理

1. 创建部门目录：`D:\Departments\HR`
2. 为 HR 团队成员授予权限
3. 其他部门无访问权限
4. 实现部门间文件隔离

### 场景 3：客户文件交付

1. 为每个客户创建独立账户
2. 创建客户目录：`D:\Clients\ClientA`
3. 仅授予该客户访问权限
4. 上传交付文件
5. 客户登录后下载

### 场景 4：备份归档

1. 创建备份目录：`D:\Backups`
2. 授予管理员读写权限
3. 授予普通用户只读权限
4. 定期上传备份文件

## ❓ 常见问题

### Q: 忘记密码怎么办？

A: 管理员可以在"用户管理"中重置用户密码。如果是管理员忘记密码，需要修改数据库：

```bash
python
>>> from app import db, User
>>> from auth import hash_password
>>> user = db.query(User).filter(User.username == 'admin').first()
>>> user.password_hash = hash_password('NewPassword123')
>>> db.commit()
```

### Q: 如何限制用户上传文件大小？

A: 编辑 `config.py`，修改 `MAX_CONTENT_LENGTH`：

```python
MAX_CONTENT_LENGTH = 100 * 1024 * 1024  # 100MB
```

### Q: 如何添加新的文件类型？

A: 编辑 `config.py`，在 `ALLOWED_EXTENSIONS` 中添加：

```python
ALLOWED_EXTENSIONS = {'txt', 'pdf', 'png', 'jpg', 'psd'}  # 添加 psd
```

### Q: 文件存储在哪里？

A: 默认存储在 `file_manager/data/users/` 目录下。可以在 `config.py` 中修改 `DATA_ROOT`。

### Q: 如何备份数据？

A: 备份以下两个位置：
1. 数据库：`file_manager.db`
2. 用户文件：`file_manager/data/users/`

```bash
# 创建备份
cp file_manager.db file_manager.db.bak
tar -czf users_backup.tar.gz data/users/
```

### Q: 支持移动端访问吗？

A: 支持！系统采用响应式设计，可在手机/平板浏览器访问。

### Q: 如何查看谁删除了文件？

A: 管理员进入"审计日志"，筛选操作类型为"删除"，即可查看所有删除记录。

## 📊 权限说明

### 权限级别

| 级别 | 可执行操作 |
|------|-----------|
| **只读** | 浏览、下载 |
| **读写** | 浏览、下载、上传、删除、重命名、创建目录 |

### 权限继承

- ✅ 子目录自动继承父目录权限
- ✅ 权限检查递归进行
- ✅ 多个权限取最高级别

### 权限示例

```
用户 A 对 D:\Documents 有读写权限
↓ 自动继承
用户 A 对 D:\Documents\Project 有读写权限
用户 A 对 D:\Documents\Project\2026 有读写权限
```
