## [1.0.0] - 2026-03-15

### ✨ 第一阶段完成

#### 新增功能
- 🔐 **多用户认证系统**
  - 用户登录/登出
  - 会话管理
  - 角色权限（管理员/普通用户）

- 📁 **权限管理系统**
  - 按目录分配读/写权限
  - 可视化权限配置界面
  - 快速授权/撤销功能
  - 支持子目录权限继承

- 📂 **文件操作**
  - 文件上传（最大 1GB）
  - 文件下载
  - 文件/目录删除
  - 目录创建
  - 文件/目录重命名
  - 树形目录导航

- 📊 **审计日志**
  - 记录所有关键操作
  - 支持按用户/操作类型筛选
  - 时间范围查询

- 👥 **用户管理**
  - 创建/编辑/删除用户
  - 密码重置
  - 用户状态管理

- 🎨 **UI 设计**
  - 科技感深色主题
  - 响应式布局
  - 移动端适配
  - Bootstrap 5.3 组件

#### 技术实现
- Flask 2.3+ 后端框架
- SQLite 数据库
- Bootstrap 5.3 前端框架
- JavaScript 原生 AJAX
- Werkzeug 密码哈希

#### 修复问题
- 🔧 修复权限撤销接口参数不匹配问题（前端发送 `permission_id`，后端期望 `directory_path`）
- 🧹 清理测试代码和冗余 UI 元素

## 📝 版本说明

### 版本命名规则
采用语义化版本号：`主版本。次版本.修订版`

- **主版本**：不兼容的 API 变更
- **次版本**：向后兼容的功能新增
- **修订版**：向后兼容的问题修复

### 符号说明
- `✨` 新增功能
- `🔧` 修复问题
- `📖` 文档更新
- `🎨` UI/UX改进
- `⚡` 性能优化
- `🔒` 安全相关
