## 版本说明

### 语义化版本

- **MAJOR.MINOR.PATCH** (主版本号。次版本号。修订号)
  - **MAJOR**: 不兼容的 API 变更
  - **MINOR**: 向后兼容的功能新增
  - **PATCH**: 向后兼容的问题修复

### 更新类型

- `Added` - 新增功能
- `Changed` - 现有功能的变更
- `Deprecated` - 即将移除的功能
- `Removed` - 已移除的功能
- `Fixed` - Bug 修复
- `Security` - 安全性修复
